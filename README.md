# Painel de Candidaturas

Ferramenta que **busca vagas de emprego, avalia cada uma e escreve seu currículo e carta de apresentação sob medida** — tudo por botões, sem precisar digitar comandos.

## 🖱️ Como usar (é só clicar)

1. Baixe este projeto para o seu computador.
2. **Dê duplo-clique em `Abrir Painel.bat`** (este arquivo, aqui nesta pasta).
3. Uma janela preta abre (o motor do painel — **deixe-a aberta**) e o painel abre sozinho no seu navegador.
4. No painel: **envie seu currículo**, **escreva o cargo** que procura e clique em **Buscar novas vagas**.

O painel encontra as vagas, dá uma nota de 0 a 100 para cada uma e prepara os documentos. O **envio final** de cada candidatura você confirma no site da empresa.

## ⚙️ Antes do primeiro uso (uma vez, por alguém com prática)

O painel depende de alguns programas instalados na máquina: **Claude Code** (logado), **Node.js**, **Bun** e **LaTeX (MiKTeX)**. Passo a passo em [`painel/LEIA-ME.txt`](painel/LEIA-ME.txt).
