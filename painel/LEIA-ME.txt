========================================================
  PAINEL DE CANDIDATURAS — como usar
========================================================

PARA O DIA A DIA (usuário)
--------------------------------------------------------
1. Dê DUPLO-CLIQUE no arquivo "Abrir Painel.bat" que está na
   RAIZ do projeto (a pasta principal, um nível acima desta).
2. Uma janela preta abre (é o motor do painel) e, em seguida,
   o painel abre sozinho no seu navegador.
3. Use os botões. Deixe a janela preta aberta enquanto usa.
4. Para fechar tudo, feche a janela preta.

O painel NÃO envia nada sozinho: ele encontra as vagas,
avalia e escreve o currículo e a carta. O clique final de
"enviar" no site da empresa é sempre seu.

DICA - Fazer várias vagas de uma vez:
Cada vaga na lista tem uma caixinha (checkbox). Marque as que
quiser (ou use "selecionar todas" no topo da lista) e aparece
uma barra com duas opções:
 - "Ver o que falta e reavaliar": mostra o que dá para acrescentar
   ao seu currículo e reavalia TODAS as marcadas de uma vez.
 - "Preparar candidaturas": monta currículo e carta de todas as
   marcadas, uma após a outra.

DICA - Ver/editar seu currículo:
Na seção "Meu currículo (por vaga)" você escolhe um currículo
já gerado e pode: Ver PDF, Baixar PDF, ou Editar texto (mexer
no conteúdo e clicar em "Salvar e recompilar" para gerar o PDF
novo). Se algo der errado na edição, o PDF anterior é mantido.

DICA - Respostas do formulário (pretensão, etc.):
Quando o site (Gupy) pedir campos como "Pretensão salarial",
"Última remuneração" ou perguntas específicas da vaga, use:
 - "Meus dados para formulários" (na tela inicial): preencha
   uma vez seus valores e o painel reusa nas próximas vagas.
 - "Respostas do formulário" (ao abrir uma vaga): ajuste e
   adicione as perguntas daquela vaga. Cada resposta tem um
   botão COPIAR para você colar no site. Fica tudo salvo.


INSTALAÇÃO (feita UMA VEZ, por alguém com um pouco de prática)
--------------------------------------------------------
Ao abrir o "Abrir Painel.bat", a janela preta CONFERE e mostra
quais destes programas estão instalados. O painel baixa sozinho
as dependências do próprio projeto (as ferramentas de busca),
mas os 4 programas abaixo precisam ser instalados por você,
pois exigem administrador e/ou login:

  1. Node.js  (obrigatório)
     Baixe e instale: https://nodejs.org  (versão 18 ou maior)

  2. Bun  (obrigatório)
     No PowerShell, cole e rode:
        powershell -c "irm bun.sh/install.ps1 | iex"

  3. Claude Code  (obrigatório, precisa estar LOGADO)
     Instale: https://claude.com/claude-code
     Depois, no terminal, rode  claude  e faça login numa conta
     com créditos/assinatura.

  4. LaTeX / MiKTeX  (para gerar os PDFs do currículo)
     Baixe e instale: https://miktex.org

  5. Git  (OPCIONAL — só para receber avisos de atualização)
     Baixe: https://git-scm.com
     Com o Git instalado, toda vez que você abrir o painel ele
     avisa se há uma versão nova no GitHub e oferece baixar.
     Sem o Git, o painel funciona igual, só não avisa de updates.

Para conferir se o Claude Code está acessível, abra o
Prompt de Comando e digite:  claude --version
Se aparecer um número de versão, está pronto.

DEPOIS de instalar os 4, é só dar duplo-clique no "Abrir Painel.bat"
de novo — na primeira vez ele baixa as dependências do projeto
(mostrando o progresso) e abre o painel.

Se o comando "claude" não for reconhecido, descubra o caminho
completo do executável e edite a 1ª linha de configuração no
"server.js": troque
        const CLAUDE_BIN = process.env.CLAUDE_BIN || "claude";
pelo caminho, por exemplo:
        const CLAUDE_BIN = "C:\\Users\\SEU_USUARIO\\.local\\bin\\claude.exe";


SE APARECER "workspace has not been trusted" (pasta não autorizada)
--------------------------------------------------------
Isso costuma acontecer UMA vez, depois de mover, renomear ou
copiar a pasta para outro computador. NÃO é erro seu e nada
foi perdido. O painel mostra um aviso em português com o botão
"Autorizar e liberar" — clique nele e pronto. Depois, é só
clicar de novo no botão da ação que você queria (buscar,
avaliar, etc.).

Se preferir resolver na mão: abra o programa "Claude Code"
uma vez dentro desta pasta e responda "Sim, confio" à pergunta.


OBSERVAÇÕES
--------------------------------------------------------
- O painel roda 100% local (endereço 127.0.0.1:4599). Nada
  é publicado na internet; ninguém de fora acessa.
- Cada ação consome créditos da sua conta Claude, porque é o
  Claude que faz o trabalho pesado.
- As ações podem levar alguns minutos (o assistente pesquisa,
  escreve e compila os documentos). É normal.
